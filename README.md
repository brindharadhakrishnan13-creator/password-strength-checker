# Password Strength Checker

## Description

A Python program that checks the strength of a password based on:

- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Features

- Weak password detection
- Medium password detection
- Strong password detection

## Technologies Used

- Python

## How to Run

```bash
python main.py
```

## Example

Input:

Hello123@

Output:

Strong Password
